# Carpeta de Imágenes de Cotizaciones

Esta carpeta contiene las imágenes subidas por los clientes y el administrador para las cotizaciones de servicios.

## Estructura

- Cada cotización tiene su propia subcarpeta identificada por el `id_cotizacion`
- Las imágenes se guardan con nombres únicos generados automáticamente
- Formato: `imagenes_cotizaciones/{id_cotizacion}/{nombre_imagen}.{extension}`

## Uso

- **Clientes:** Suben imágenes al solicitar una cotización desde el formulario público
- **Administrador:** Puede subir imágenes adicionales al crear un nuevo servicio desde el panel de administración

Ambos tipos de imágenes se almacenan en la misma ubicación para facilitar su gestión y visualización.
