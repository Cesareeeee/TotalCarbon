# Carpeta de Imágenes del Proceso de Reparación

Esta carpeta contiene las imágenes subidas por el administrador durante el proceso de reparación de las bicicletas.

## Estructura

- Cada cotización tiene su propia subcarpeta identificada por el `id_cotizacion`
- Las imágenes se guardan con nombres personalizados o generados automáticamente
- Formato: `imagenes_proceso/{id_cotizacion}/{nombre_imagen}.{extension}`

## Uso

Las imágenes son subidas desde el panel de administrador en la sección "Seguimiento de Proyectos" y se muestran tanto al administrador como al cliente en sus respectivos paneles.
